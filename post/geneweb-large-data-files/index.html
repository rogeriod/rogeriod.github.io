GeneWeb - Large data files

  Rogério Domingos

##Geneweb - large data files

Ficheiros de grande dimensão – tipo JC com 65,000 indivíduos – falham quando se importa directamente do GED.

Se proveniente de um PAF, “limpar” o ficheiro:

  1. importar para o PAF, verificar consistência dos dados e reexportar como GED.
  2. find/replace 2 NPFX com 1 TITL.
  3. importar para o Geneanet (que usa o GW) e reparar eventuais erros detectados durante a importação.
  4. exportar árvore em formato GW.
  5. fazer upload dos ficheiros basexxx.gw e basexxx.gwf para o servidor e construir bases apartir do ficheiro basexxx.gw
